About

This is a search website for ✨The Wandering Inn✨, a great fantasy LitRPG webnovel. It's got memorable characters, expansive worldbuilding, and a gripping story full of magic and monsters. You should check it out!

I started developing this site while the webnovel's own search function was broken. It has since been fixed, but I figured I could improve the UI/UX so I built this site anyways. Here are some features:

  • Each search result shows excerpts containing highlighted keywords. For chapters with more than three excerpts, you can click to expand the result box and see all occurrences in that chapter.
  • You can sort the search results by relevance or by chronology. Be careful sorting by "Latest" if you want to avoid spoilers!
  • As is usual for search functions, you can wrap a phrase in quotes to search for that phrase as a whole rather than its constituent keywords. I might add other search features like specifying required keywords if users express interest.
  • A few Easter eggs...?

The souce code is public on Github. If you find any bugs, usability issues, or features you'd like to see, join the TWI Discord and ping me @rocketscience.
